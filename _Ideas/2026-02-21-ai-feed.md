## Summary

## Details

### Sources (all free, no auth)
| Source | URL | Content Type |
|--------|-----|--------------|
| r/MachineLearning | reddit.com/r/MachineLearning/.rss | Research, papers |
| r/artificial | reddit.com/r/artificial/.rss | General AI news |
| r/LocalLLaMA | reddit.com/r/LocalLLaMA/.rss | Open source models |
| Hacker News | hnrss.org | Tech discussions |

| DEV.to | dev.to/feed/tag/ai | Developer articles |
| ArXiv | export.arxiv.org/rss/cs.AI | Research papers |

### Features
- **Sorted by popularity**: Extracts upvotes/points, shows most popular first
- **AI trend summary**: LLM analyzes top posts to identify trending topics
- **No API keys**: Uses RSS feeds only

### Why RSS instead of APIs
- Twitter/X API: $100+/month, requires approval
- Reddit API: Now requires approval process
- RSS feeds: Free, instant, no auth needed

## Implementation
- Uses `feedparser` (already a dependency)
- Extracts scores from Reddit/HN RSS content
- LLM summarizes trends from top post titles
